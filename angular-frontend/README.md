# Angular Frontend

Frontend Angular de la aplicacion. La documentacion principal esta en el `README.md` de la raiz del repositorio.

## Comandos

Instalar dependencias:

```bash
pnpm install
```

Levantar servidor local:

```bash
pnpm start
```

Ejecutar tests:

```bash
pnpm test -- --watch=false
```

Compilar:

```bash
pnpm run build
```
